## Problem Statement

Software project built by autocoder.cc
The platform “Tasky” help with managing and tracking tasks, it allows you to add new tasks, with name, priority, deadline,
and optional description, and category. It also keeps the user motivated by showing: the total tasks, how many completed tasks,
pending tasks, and overdue tasks.

## Features

- **Dashboard** — stats: total tasks, completed, pending, and overdue counts
- **Task Board** — Add, delete, and mark tasks complete with instant visual feedback
- **Task Fields** — Title, priority level (Low / Medium / High), due date, category tag
- **Priority Color Coding** — Green (Low) · Yellow (Medium) · Red (High)
- **Categories Page** — View and manage tags with per-category task counts
- **Dark Theme** — Purple and violet accent palette with smooth CSS animations
- **Card Layout** — Modern, responsive card-based UI throughout all pages

### Option 2 — Regenerate on autocoder.cc

1. Go to **https://autocoder.cc** and create a free account
2. Click **New Project**
3. Paste the following prompt exactly:

```
Build a modern task management web app called Tasky. It should have 3 pages:
a Dashboard showing task statistics (total, completed, pending, overdue),
a Task Board where users can add, delete, and mark tasks as complete,
and a Categories page to manage tags. Each task should have a title,
priority level (Low / Medium / High with color coding: green/yellow/red),
a due date, and a category tag. Tasks should persist using local storage.
Use a sleek dark theme with purple and violet accents, smooth animations,
and a modern card-based layout.
```

4. Click Generate and wait until 10 minutes
5. Your live URL is displayed immediately — the app is accessible without any further setup
